## rust 编译过程

1. rust code(UTF-8) -> 分词 ->
2. Tokens —> 解析 ->
3. AST -> 降级 ->
4. HIR(高级中间语言)：进行类型检查，方法查找等工作 -> 降级 ->
5. MIR(中级中间语言)：借用检查、优化、代码生成、宏、范型、单态化 -> 优化 ->
6. LLVM(低级中间语言)：生成机器码 -> 优化 ->
7. 机器码
