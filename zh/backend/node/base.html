  console.log('data', data.toString())

# buffer

缓冲区：开辟了一段内存的区域。 出现原因，因为前期设计中，无对二进制数据的处理。相当于在内存中开启了一块二进制的缓冲区。

// 创建一块buffer
const buf1 = Buffer.alloc(10)

// 读取字符串
const buf2 = Buffer.from("a")
console.log(buf2)

// 读取中文
// 创建Buffer包含的语言
const buf3 = Buffer.from("中文")
console.log(buf3)

// buffer链接
const buf4 = Buffer.concat([buf2, buf3]);
